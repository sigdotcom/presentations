Interfaces

  • A lot like classes, but only exist at compile time
  • Used to assert structural equivalence on compile

React 101

  • declarative, efficient, and flexible library for building user interfaces
  • Simply a 'renderer' (paints the page the way you want it)

JSX

React's very powerful 'templating language' that provides syntactic sugar for Reacts underlying functionality.

Elements

Elements are the basic building blocks of react (think of HTML elements).

Components

Reusable groups of Elements that accept arbitrary input and can run internal logic.

Class vs Functional Components

Two different ways of expressing components due to how React was developed. Prefer functional components.

Component Lifecycle

Events

Handling DOM events using React such as 'onClick'

Conditional Rendering

Since components are equivalent to functions, you can decide what you want to render on the fly (based on state, props, etc.)

Rendering Lists

Using special syntax, React can efficiently update lists of items (only rendering the items that change).
