What is Python?

An interpreted, general use, dynamically typed programming language with automatic memory management and support for multiple programming paradigms with whitespace indentation.

Why use Python?

  1. Easy to read
  2. Easy to write
  3. Rapid Prototyping

What is Python used for?

  1. Web Development: Django, Pyramid, Bottle, Tornado, Flask, web2py
  2. GUI Development: tkInter, PyGObject, PyQt, PySide, Kivy, wxPython
  3. Scientific and Numeric: SciPy, Pandas, IPython
  4. Software Development: Buildbot, Trac, Roundup
  5. System Administration: Ansible, Salt, OpenStack

Built-in Types

int()

Integers

              
              x = 0
              x = 10
              x = int("10")

float()

Floating Point Numbers

              
              x = 1.0
              x = 10.0
              x = float("10.0")

str()

Strings; No char type

              
              x = "test"
              x = '10'
              x = str(15)

              # Concatenation
              x = 'Hello '
              y = ' World!'
              print(x + y) # "Hello World!"

              # Repeating
              print("Hi"*5) # "HiHiHiHiHi"

              # Formatting
              x = "{}".format("Hello")
              print(x) # "Hello"

              x = "3 * 2 = {}".format(3*2)
              print(x) # '3 * 2 = 6'

bool()

Boolean - True or False

              
              x = True
              x = False
              x = bool(1) # True
              x = bool(0) # False

tuple()

  • Iterable, immutable sequence of elements.
              
              x = tuple()
              x = ()
              x = (1, 2, 3)
              x = ("test", 1)
              x[0] # "test"
              x[0] = "foo" # TypeError

list()

  • Iterable, mutable sequence of elements.
  • Can grow to "infinite" length
              
              x = list()
              x = []
              x = ["test", "test"]
              x = [1, 3]
              x = ["test", 1]
              x[0] = 3
              x.append("test")
              print(x) # [3, 1, 'test']

List Indexing

Regular
              
              x = ["foo", "bar", 10]
              x[0] # "foo"
              x[2] # 10
              x[3] # IndexError
Negative
              
              x = ["foo", "bar", 10]
              x[-1] # 10
              x[-2] # "bar"
              x[-4] # IndexError

List Slicing

              
              sliceable[start:stop:step]
  • Returns a shallow copy of the 'spliced' list
  • start - Where to begin slicing, inclusive
  • stop - Where to end slicing, exclusive
  • step- How far to 'step' each time

List Slicing (ex.)

              
              x = [10, "test", 30, "apple"]

              x[0:]   # [10, 'test', 30, 'apple']
              x[1:2]  # ['test']
              x[1::2] # ['test', 'apple']
              x[::-1] # ['apple', 30, 'test', 10]

dict()

  • Iterable, mutable collection of key-value pairs
  • Order of keys is not guaranteed
              
              x = dict()
              x = {}
              x = {"test": 10, "apple": 10}
              x = {10: "test", "foo": "bar"}

              x[10] # "test"
              x["not_real"] # KeyError

Logical Operators

              
              True and True # True
              False or False # False

Membership Operator

  • in operator
  • Returns true if the "subject" is a member of the "predicate", otherwise false 
              
              x = [1,2,3,4]

              4 in x # True
              50 in x # False

              y = {"a": 10, "b": 50}
              50 in y # False
              50 in y.values() # True

Identity Operator

  • is operator
  • Returns True if the "subject" has the same identity as the "predicate", otherwise False
  • DANGER!
              
              x = [1,2,3,4]
              y = [1,2,3,4]
              x == y # True
              x is y # False

              x = 10
              x is None # False
              x = None
              x is None # True

Flow Control

If...Elif...Else

  • Takes a boolean
  • Colon after statement; no more curly brackets
              
              x = 3

              if(x == 0):
                  print("x == 0")
              elif(x < 0):
                  print("x < 0")
              elif(x > 0):
                  print("x > 0")
              else:
                  print("x not == 0 or not < 0 or not > 0")

For Loop

  • Takes an iterable that can 'yield' one member at a time
    1. list
    2. str
    3. tuple
    4. dict
    5. File Object

For Loop (cont.)

              
              >> x = [1,2,3]

              >> for item in x:
              >>    print(x)
              >>    ...
              1
              2
              3

For Loop (subcommands)

  • pass
  • break
  • continue
  • else

Subcommands (ex.)

              
              lst = [1,2,3,4,5]

              for x in lst:
                  if x == 2:
                      continue
                  elif x > 4:
                      break
                  else:
                      print(x)
              else:
                  print("Looped Successfully")

Functions

Syntax

            
            def function_name(arguments):
                ...

            # Example
            def is_even(num):
                return num % 2 == 0

Calling a function

            
            def is_even(num):
                return num % 2 == 0

            is_four_even = is_even(4)
            print(is_four_even) # True

Some Notes

  • All parameters are passed by object reference
  • If no return type is specified, None is returned
  • It is possible to return more than one item

Argument Types

  • Positional Arguments
  • Keyword Arguments
            
            def is_evenly_divisible(num, divisor=2):
                return num % divisor == 0
            
            
            
            def cheeseshop(kind, *arguments, **keywords):
                print("-- Do you have any", kind, "?")
                print("-- I'm sorry, we're all out of", kind)
                for arg in arguments:
                    print(arg)
                print("-" * 40)
                keys = sorted(keywords.keys())
                for kw in keys:
                  print(kw, ":", keywords[kw])

Classes

Syntax

Classes Example

            
            class Car(Object):
                """Describes a car"""
                color = red

                def __init__(self, model):
                    self.model = model 

                def get_color(self):
                    return self.color

                def set_color(self, num):
                    self.color = num

Classes Example (cont.)

            
            ford_escape = Car("Ford") # Class Instantiation
            ford_escape.get_fuel() # 100
            ford_escape.get_color() # red
            ford_escape.color # red
            ford_escape.model # Ford
